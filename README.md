# test-thortech-crud-nest

Thortech coding test, crud with NestJS + TelegramBot

## Installation

```bash
$ npm install
```

## Run

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Guide

- Open swagger document

```bash
http://localhost:3000
```

- If u wanna see list chatId / username that join our bot telegram

```bash
https://api.telegram.org/bot<TOKEN>/getUpdates
```
